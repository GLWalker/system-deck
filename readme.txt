Admin workspace helper.

== Description ==

SystemDeck is a high-performance admin workspace helper that provides a drawer interface for easy access to widgets and tools.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/system-deck` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->SystemDeck screen to configure the plugin.
